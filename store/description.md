## Detailed description
Chatter Ghost is an AI assistant for professional chatters managing creator accounts on subscription platforms.

When a fan sends a message, Chatter Ghost instantly generates 3 contextual reply suggestions directly inside the chat interface — no copy-pasting, no tab switching.

**Features:**
- 3 reply suggestions per message (Engage / Soft Sell / Direct Sell)
- 3 conversation modes: Warm Up, Sell, Re-engage
- Fan profile tracking — spend tier, subscription duration, buying power score
- Auto-tags fans: whale, ppv buyer, time waster, ghosted
- AI translation — translate any suggestion to the fan's language with one click
- Drag to reposition and resize the suggestion panel
- Per-fan mode memory — each conversation remembers its last mode
- Keyboard shortcuts: Alt+1/2/3 to insert, Alt+R to regenerate
- Works across multiple creator accounts

**How it works:**
Chatter Ghost connects to the Anthropic Claude API using your own API key. All fan data is stored locally in your browser — nothing is sent to any third-party server except the Anthropic API for generating suggestions.

**Requirements:**
- An Anthropic API key (get one at console.anthropic.com)
- A subscription platform account with active conversations
